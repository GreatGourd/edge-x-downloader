# X 媒体下载助手

Edge 浏览器扩展 — 在 X(Twitter) 上方便地下载图片和视频。

## 功能

- 点击推文的**书签按钮** → 自动收藏 + 弹出"另存为"下载对话框
- **图片**和**视频**可设置不同的默认保存路径
- 浏览器会记住上次使用的目录
- 无需额外 UI，无缝集成到 X 原生操作

## 安装

1. 下载最新的 [Release](https://github.com/GreatGourd/edge-x-downloader/releases)
2. 解压到本地文件夹
3. 打开 Edge → `edge://extensions/`
4. 开启「开发人员模式」
5. 点击「加载解压缩的扩展」→ 选择解压后的文件夹

## 使用

1. 在 X 刷到带图片/视频的推文
2. 点击推文右下角的 📑 **书签按钮**
3. → 推文正常收藏 ✓ + 自动弹出「另存为」对话框
4. 选择保存位置即可

## 路径设置

右键扩展图标 → **选项**，可分别设置图片和视频的默认保存路径。

## 原理

通过拦截 X 的 GraphQL API 响应，从 `video_info.variants` 中提取最高码率的 MP4 直链，实现高质量视频下载。
